SETUP:
take the directory of this folder and make an environment variable called "SusOrb" then go to "code\__pycache__" click on the MCOABS3D.cpython-39.pyc and enjoy the game!

MODDING:
simply edit "code\MCOABS3D.py" then open command prompt do "pip install pyinstaller" then do ' cd "Desktop\Mogus Capture Orb And Become Sus 3D\code" ' and then finally do "pyinstaller MCOABS3D.py" and when it asks you if you want to 
continue with options (y/N) type "y" and hit enter

requirements:
python
a code editor
"SusOrb" environment variable
https://www.panda3d.org/ (Panda3D)
https://sourceforge.net/projects/directpython11/files/ (DirectPython)
https://github.com/Moguri/panda3d-gltf (gltf)
